# Blog Posts

This directory contains the published HTML blog posts for the [Shailesh Mishra](https://sendtoshailesh.github.io/content-creation/) GitHub Pages site.

## Structure

```
blog/
├── *.html          # Published blog post pages
└── visuals/        # Visual assets (PNGs, SVGs) referenced by blog posts
```

## Publishing

Blog posts are published by the `@web-publisher` agent, which converts completed Markdown blog posts from `content/` into HTML and links them from the site index.

See the [content pipeline README](../../README.md) for the full publishing workflow.
