## Usage Notes

### LinkedIn Carousel
- Use slides 01–10 in order as a single multi-image post
- Slide 01 (hook) is the cover — visually grab attention with the "120x" number
- Slide 08 (pattern interrupt) breaks the informational flow — leave it in position
- Add the canonical URL in the post copy, NOT embedded in the image

### X / Twitter Thread
- Post `x-card-01.png` with tweet 1 (the hook tweet)
- Post `x-card-02.png` with the data/evidence tweet
- Post `x-card-03.png` with the framework explanation tweet
- Post `x-card-04.png` as the final CTA tweet

### Medium Article
- Use `medium-hero.png` as the cover image
- Embed `medium-inline-01.png` after the "What Changed" section
- Embed `medium-inline-02.png` after the case study paragraph

### Substack Note
- Use `substack-hero.png` as the note's lead image
- 150–300 character copy only; link to canonical URL

### LinkedIn Article
- Use `linkedin-article-exhibit-01.png` and `linkedin-article-exhibit-02.png`
- Must include >30% original content not in the blog post to avoid SEO canonical conflict
